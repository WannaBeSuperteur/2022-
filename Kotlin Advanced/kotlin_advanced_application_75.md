## 요구사항 분석
다음의 계산식을 만족시키는, 10보다 크지 않은 음이 아닌 정수 ```a```, ```b```, ```c```의 값의 순서쌍을 모두 찾아서 출력하는 함수 ```printSolutions(formula)```를 작성하시오.

계산식 예시:
```kotlin
a + bc = ab + c
```

출력 예시:
```kotlin
(a, b, c) = (0, 0, 0)
(a, b, c) = (1, 1, 1)
(a, b, c) = (1, 2, 1)
...
```

단, 정렬 순서는 출력 결과의 각 정수의 값들을 기준으로 오름차순이며, 정수는 a, b, c가 아닐 수도 있다. 또한, 연산 기호로는 ```+```, ```*```(생략) 의 2가지만 사용할 수 있다.
