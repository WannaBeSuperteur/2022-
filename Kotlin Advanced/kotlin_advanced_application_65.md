## 요구사항 분석
오름차순으로 정렬된 주어진 배열 ```array```에서, ```N```진 탐색을 이용하여 값 ```value```를 찾는 과정을 나타내는 함수 ```nSearch(array, N, value)```를 작성하시오.

예를 들어 ```[5, 10, 12, 14, 15, 20, 26, 28, 29, 32, 35, 36, 40, 43, 44, 47, 51, 59, 67, 71]```에서 3진 탐색을 이용하여 15를 찾는 과정은 다음과 같다.

```kotlin
기준점 : 26, 43
찾을 값 15는 1번째 기준점의 값 26보다 작으므로 [5, ..., 20] 을 탐색한다.
기준점 : 10, 14
찾을 값 15는 2번째 기준점의 값 14보다 크므로 [15, 20] 을 탐색한다.
기준점 : 15, 20
찾을 값 15는 1번째 기준점의 값 15와(과) 같으므로 찾을 값이 위치한 인덱스는 4이다.
```

단, 기준점과의 값 비교는 ```1번째 기준점의 값 ~보다 작으므로```, ```k번째 기준점의 값 ~와(과) 같으므로```, ```k번째 기준점의 값 ~보다 크고 (k+1)번째 기준점의 값 ~보다 작으므로```, ```(N-1)번째 기준점의 값 ~보다 크므로``` 중 하나로 나타낸다.

## 코드 및 실행 결과
### 코드
```kotlin
// 배열을 [A], [A, B] 또는 [A, ..., B] 형식으로 출력하기
fun printArray(array: List<Int>): String {
    return when (array.size) {
        0 -> "[]"
        1 -> "[${array[0]}]"
        2 -> "[${array[0]}, ${array[1]}]"
        else -> "[${array[0]}, ..., ${array[array.size - 1]}]"
    }
}

// 값 찾기
fun nSearch(array: List<Int>, N: Int, value: Int) {
    var currentArray = array
    var startIndex = 0
    
    while (true) {
        
        // 기준점의 인덱스
        val pointIndices = (1 until N).map {
            currentArray.size * it / N
        }
        val points = pointIndices.map {
            currentArray[it]
        }
        println("기준점 : ${points.joinToString(", ")}")
        
        // 기준점 중 일치하는 것이 있으면
        var isMatched = false
        
        pointIndices.forEachIndexed { idx, it ->
            if (!isMatched && currentArray[it] == value) {
                println("찾을 값 ${value}는 ${idx + 1}번째 기준점의 값 ${value}와(과) 같으므로 찾을 값이 위치한 인덱스는 ${startIndex + it}이다.\n")
                
                isMatched = true
                return@forEachIndexed
            }
        }
        
        if (isMatched) {
            return
        }
        
        // 그렇지 않으면
        val countLargerThanPoints = points.count { value > it }
        
        when (countLargerThanPoints) {
            
            // 첫 번째 기준점보다 작음
            0 -> {
                currentArray = currentArray.slice(0 .. pointIndices[0] - 1)
                print("찾을 값 ${value}는 1번째 기준점의 값 ${points[0]} 보다 작으므로 ")
            }
            
            // 마지막 기준점보다 큼
            N - 1 -> {
                currentArray = currentArray.slice(pointIndices[N - 2] + 1 .. currentArray.size - 1)
                print("찾을 값 ${value}는 ${N - 1}번째 기준점의 값 ${points[N - 2]} 보다 크므로 ")
                
                startIndex += pointIndices[N - 2] + 1
            }
            
            // 일반적인 경우
            else -> {
                currentArray = currentArray.slice(pointIndices[countLargerThanPoints - 1] + 1 .. pointIndices[countLargerThanPoints] - 1)
                print("찾을 값 ${value}는 ${countLargerThanPoints}번째 기준점의 값 ${points[countLargerThanPoints - 1]}보다 크고 ")
                print("${countLargerThanPoints + 1}번째 기준점의 값 ${points[countLargerThanPoints]}보다 작으므로 ")
                
                startIndex += pointIndices[countLargerThanPoints - 1] + 1
            }
        }
        println("${printArray(currentArray)} 을 탐색한다.")
    }
}

fun main() {
    listOf(2, 3, 4).forEach {
        nSearch(listOf(5, 10, 12, 14, 15, 20, 26, 28, 29, 32, 35, 36, 40, 43, 44, 47, 51, 59, 67, 71), it, 15)
        nSearch(listOf(5, 10, 12, 14, 15, 20, 26, 28, 29, 32, 35, 36, 40, 43, 44, 47, 51, 59, 67), it, 32)
        nSearch(listOf(5, 10, 12, 14, 15, 20, 26, 28, 29, 32, 35, 36, 40, 43, 44, 47, 51, 59), it, 36)
        nSearch(listOf(5, 10, 12, 14, 15, 20, 26, 28, 29, 32, 35, 36, 40, 43, 44, 47, 51), it, 12)
        nSearch(listOf(5, 10, 12, 14, 15, 20, 26, 28, 29, 32, 35, 36, 40, 43, 44, 47), it, 44)
        nSearch(listOf(5, 10, 12, 14, 15, 20, 26, 28, 29, 32, 35, 36, 40, 43, 44), it, 28)
    }
}
```
