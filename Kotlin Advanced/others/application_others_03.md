## 요구사항 분석
1000부터 1260까지의 각 자연수에 대해서 다음을 구하시오.
* 약수의 개수
* 약수의 합
* 약수가 12개 이상인 약수의 개수

또한, 위의 3가지 값 중 '약수의 개수' 및 '약수의 합'이 각각의 특정 값으로 나타나는 첫 자연수를 구하시오. (예: 6, 8, 10의 약수의 개수는 4개이므로, '약수의 개수'가 4인 첫 자연수는 6이다.)

## 코드
```kotlin
data class Number (
    var value: Int = 0
)

val Number.divisors: Int get() = (1..this.value).count { this.value % it == 0 }
val Number.divisorList: List<Int> get() = (1..this.value).filter { this.value % it == 0 }
val Number.divisorSum: Int get() = Number::divisorList.get(this).sum()

fun main() {
    fun isDivisor12(num: Number) = Number::divisors.get(num) >= 12
    
    // reflection을 이용
    for (i in 1000..1260) {
        val n = Number(i)
        println("\n1. 수: ${i}, 약수 개수: ${String.format("%02d", n.divisors)}, 합: ${n.divisorSum}")
        
        val divisors = Number::divisorList.get(n)
        val divisors12 = divisors.map { Number(it) }.filter(::isDivisor12).map { it.value }
        println("2. 약수 목록: ${divisors}")
        println("3. 약수가 12개 이상인 약수 (${divisors12.size}개) 목록: ${divisors12}")
    }

    // 1260 ~ 1000으로 하기 위하여 asReversed() 적용
    val numbers = (1000..1260).map { Number(it) }.asReversed()
    println("\n4. numbers = ${numbers}")
    
    // associateBy() 적용
    val divisorsCount = numbers.associateBy { it.divisors }
    val divisorsSum = numbers.associateBy { it.divisorSum }
    
    print("\n")
    divisorsCount.forEach { key, value ->
        println("5. 약수의 개수가 ${key}인 최초의 정수: ${value.value}")
    }
    
    print("\n")
    divisorsSum.forEach { key, value ->
        println("6. 약수의 합이 ${key}인 최초의 정수: ${value.value}")
    }
}
```
