## 요구사항 분석
주어진 자연수들을 다음의 우선순위에 따라 정렬하시오.
* 짝수를 홀수보다 먼저 나열한다.
* 짝수의 경우 다음과 같이 한다.
  * 5로 나눈 나머지가 3, 0, 2, 1, 4인 순서대로 나열한다.
  * 5로 나눈 나머지가 같은 경우 '1'의 개수가 많은 순서대로 나열한다.
  * '1'의 개수도 같은 경우 작은 순서에서 큰 순서로 나열한다.
* 홀수의 경우 다음과 같이 한다.
  * 5로 나눈 나머지가 3, 0, 2, 1, 4인 순서대로 나열한다.
  * 5로 나눈 나머지가 같은 경우 '3'의 개수가 많은 순서대로 나열한다.
  * '3'의 개수도 같은 경우 '2'의 개수가 많은 순서대로 정렬한다.
  * '2'의 개수도 같은 경우 작은 순서에서 큰 순서로 나열한다.

## 코드
```kotlin
import java.util.Random

fun sortNumberList(numberList: List<Int>): List<Int> {
    val modMap = mapOf(3 to 0, 0 to 1, 2 to 2, 1 to 3, 4 to 4)
    
    // comparator를 이용하여 짝수 및 홀수를 공통 비교
    val commonComparator = compareBy<Int> { modMap[it % 5]!! }
    val (even, odd) = numberList.partition { it % 2 == 0 }
    
    println("0. even        = ${even}")
    println("1. odd         = ${odd}")
    
    // 짝수를 정렬 (최상위 기준에 의한 정렬은 마지막에 공통 comparator를 이용)
    val evenSorted1 = even.sortedWith(compareBy(
        { it.toString().count { it == '1' } * (-1) },
        { it }
    ))
    val evenSorted2 = evenSorted1.sortedWith(commonComparator)
    
    // 홀수를 정렬
    val oddSorted1 = odd.sortedWith(compareBy(
        { it.toString().count { it == '3' } * (-1) },
        { it.toString().count { it == '2' } * (-1) },
        { it }
    ))
    val oddSorted2 = oddSorted1.sortedWith(commonComparator)
    
    println("2. sorted even = ${evenSorted2}")
    println("3. sorted odd  = ${oddSorted2}")
    
    return evenSorted2 + oddSorted2
}

fun main() {
	val random = Random()
    
    for (i in 0..99) {
        val n = random.nextInt(15) + 10
        val numberList = mutableListOf<Int>()
        
        for (j in 0..n) {
            numberList.add(random.nextInt(90) + 10)
        }
        
        println("\noriginal : ${numberList}")
        val sortedList = sortNumberList(numberList)
        println("sorted   : ${sortedList}")
    }
}
```
